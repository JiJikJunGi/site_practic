HTML이란?

HTML (Hypertext Markup Language,하이퍼텍스트 마크업 언어)는 프로그래밍 언어는 아니고,
우리가 보는 웹페이지가 어떻게 구조화되어 있는지 브라우저로 하여금 알 수 있도록 하는 마크업 언어입니다.
이는 개발자로 하여금 복잡하게도 간단하게도 프로그래밍 할 수 있습니다.
HTML은 요소(Elements)로 구성되어 있으며, 이들은 적절한 방법으로 나타내고 실행하기 위해 각 컨텐츠의 여러 부분들을 감싸고 마크업 합니다.
tags 는 웹 상의 다른 페이지로 이동하게 하는 하이퍼링크 내용들을 생성하거나, 단어를 강조하는 등의 역할을 합니다.

HTML 요소

HTML 요소(Elements)는 HTML 문서 내에서 콘텐츠를 표시하거나 구조화하기 위해 사용되는 구성 요소 입니다.
각 요소는 태그 정의되며, 여는 태그와 닫는 태그 사이에 콘텐츠가 위치합니다. ※ 여는 태그는 '<tag>'와 같이 작성되고, 닫는 태그는 '</tag>' 와 같이 작성됩니다.

HTML 문서에서 일반적으로 사용되는 몇 가지 기본 요소들은 다음과 같습니다.
  1. "<html>" : HTML 문서의 루트 요소를 정의합니다.
  2. '<head>' : 문서의 메타 정보를 포함하며, 주로 문서 제목('<title>'), 스타일 시트 링크, 메타 데이터 등을 포함합니다.
  3. '<title>' : 문서의 제목을 정의합니다. 브라우저 탭에 표시됩니다.
  4. '<body>' : 문서의 본문을 정의하며, 실제 내용을 포함합니다.
  5. '<h1>' , <h2>' , ... ,'<h6>' : 제목을 표시하는 요소로, 숫자가 클수록 제목의 수준이 낮아집니다.
  6. '<p>' : 단락(paragraph)을 나타내는 요소입니다.
  7. '<a>' : 링크(anchor)를 생성하는 요소로, 다른 웹 페이지나 파일로 연결할 수 있습니다.
  8. '<img>' : 이미지를 표시하는 요소로, 이미지 파일을 로드하여 표시합니다.
  9. '<u1>', '<o1>' , <1i>' : 비순서형 또는 순서형 목록을 정의하고, 각 목록 항목을 나타냅니다.
  10. '<div>' , '<span>' : 레이아웃을 위해 사용되는 컨테이너 요소로, CSS 스타일링이나 JavaScript 적용에 사용됩니다.
  11. '<table>' , '<tr>' , '<td>' , '<th>' : 표를 생성하는 요소들로, 행과 열을 정의하고 셀 내용을 배치합니다.
  12. '<form>' , '<input>' , '<button>' : 사용자 입력을 받는 폼을 생성하며, 텍스트 입력, 라디오 버튼, 체크박스, 버튼 등을 포함합니다.

HTML 요소(Element)의 구조

HTML에서의 요소에 대해 좀 더 알아봅시다.



엘리먼트의 주요 부분은 다음과 같습니다.
  1. 여는 태그(Opening tag): 이것은 요소의 이름과(이 경우 p), 열고 닫는 꺽쇠 괄호로 구성됩니다. 요소가 시작(이 경우 단락의 시작 부분)부터 효과가 적용되기 시작합니다.
  2. 닫는 태그(Closing tag): 이것은 요소의 이름 앞에 슬래시(/)가 있는것을 제외하면 여는 태그(opening tag)와 같습니다.
    이것은 요소의 끝(이 경우 단락의 끝 부분)에 위치합니다. 닫는 태그를 적어주지 않는 것은 흔한 초심자의 오류이며, 이것은 이상한 결과를 낳게됩니다.
  3. 내용(Content): 요소의 내용이며, 이 경우 단순한 텍스트이다.
  4. 요소(Element): 여는 태그, 닫는 태그, 내용을 통틀어 요소(element)라고한다.
